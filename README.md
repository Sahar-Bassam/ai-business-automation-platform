# AI Business Automation Platform (FireShip)

A full-stack SaaS-style web application that helps businesses explore and purchase AI-powered automation solutions such as appointment scheduling, customer support agents, analytics, and workflow automation.

Built with **Vue.js + Laravel**, this project simulates a real-world AI services platform with dynamic plans, interactive UI, and backend integration.

---

## Design Process

Before development, the UI/UX was first designed in **Figma** to plan the structure, layout, and user experience.

The initial Figma design was inspired by a tutorial/design reference from a YouTube creator, and then adapted and implemented into a fully functional web application using Vue.js and Laravel.

This helped ensure a clean design system and a more structured development workflow.

---

## Features

* Dynamic AI service marketplace
* AI plan selection based on company size
* Full form system with validation
* API integration between Vue frontend and Laravel backend
* Modern SaaS landing page UI
* Smooth GSAP animated service showcase
* Plan selection + purchase flow simulation
* Component-based architecture (Vue)

---

##  Tech Stack

**Frontend:**

* Vue.js (Composition API)
* Tailwind CSS
* GSAP (animations)
* Inertia.js

**Backend:**

* Laravel
* RESTful APIs
* MySQL / SQLite
* Laravel Events, Controllers, Notifications
